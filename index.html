Connect Mysql With Spark

Jan 29th, 2016 3:12 pm

  1. 设置SPARK_CLASSPATH,如： bash export SPARK_CLASSPATH=/home/ting/software/mysql-connector-java-5.1.38/mysql-connector-java-5.1.38-bin.jar
  2. 启动spark-shell bash sh ./bin/spark-shell
  3. 把Mysql 表转化为DataFrame对象 scala val jdbcDF = sqlContext.load("jdbc", Map("url" -> "jdbc:mysql://localhost:3306/your_database?user=your_user&password=your_password", "dbtable" -> "your_table"))
  4. DataFrame的一些简单用法 ``` // 输出表结构 df.printSchema()

// 选择所有年龄大于21岁的人，只保留name字段 df.filter(df(“age”) > 21).select(“name”).show()

// 选择name，并把age字段自增 df.select(“name”, df(“age”) + 1).show()

// 按年龄分组计数 df.groupBy(“age”).count().show()

// 左联表（注意是3个等号！） df.join(df2, df(“name”) === df2(“name”), “left”).show() ```
